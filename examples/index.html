DevDashboard

Developer Productivity Metrics Visualization Library

About DevDashboard

DevDashboard is a lightweight JavaScript library for visualizing developer productivity metrics and project data. It provides an easy way to create beautiful, interactive dashboards that display metrics like commit frequency, code quality, and sprint velocity.

These examples demonstrate how to use DevDashboard to create various types of visualizations. Feel free to explore and use the code as a starting point for your own projects.

API Endpoints

This demo server provides the following API endpoints for testing:

  • /api/data/commits?days=30 - Get commit data for the last 30 days
  • /api/data/quality?days=30 - Get code quality data for the last 30 days
  • /api/data/story-points?sprints=8 - Get story points data for 8 sprints

You can adjust the parameters to get different amounts of data.
